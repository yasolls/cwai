| Version | Supported |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Instead, contact the maintainer directly:

- Email: nikmd1306@gmail.com
- Telegram: https://t.me/nikmd1306

You can expect an initial response within 48 hours. Once the issue is confirmed, a fix will be released as soon as possible.
